# Wine Quality Predictor (PyCharm + Streamlit)

## Run in PyCharm

### Step 1: Train Model (Run Once)
1. Open `train_model.py`
2. Click **Run** (or right-click → Run)
3. Wait → Creates `model.pkl`

### Step 2: Launch App
1. Open `main.py`
2. In terminal (bottom of PyCharm):
   ```bash
   streamlit run main.py
